StringBuffer vs StringBuilder

发表于 2016-07-21   |  

StringBuffer and StringBuilder are both extends from AbstractStringBuilder.

Definition

StringBuffer

A thread-safe, mutable sequence of characters.
Since JDK1.0

StringBuilder

A mutable sequence of characters. This class provides an API compatible code StringBuffer, but with no guarantee of synchronization.
This class is designed for use as a drop-in replacement for code StringBuffer. Since JDK 1.5.
